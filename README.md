# pitel_ui_kit
### Pitel UI Kit Demo

[![N|Solid](https://documents.tel4vn.com/img/pitel-logo.png)](https://documents.tel4vn.com/)

pitel_ui_kit is demo project.

## Installation
Pitel UI Kit requires flutter version 3.7.7, dart version 2.19.4
- Setup: follow guide in [here](https://github.com/tel4vn/flutter-pitel-voip/blob/main/PUSH_NOTIF.md) to setting push notification (FCM for android), Pushkit (for IOS).
- Get package
```yaml
flutter pub get
```
- Run source code
```yaml
flutter run
```
