# pitel_ui_kit
### Pitel UI Kit Demo

[![N|Solid](https://documents.tel4vn.com/img/pitel-logo.png)](https://documents.tel4vn.com/)

pitel_ui_kit is demo project.

## Installation
Pitel UI Kit requires flutter version 3.7.7, dart version 2.19.4
- Get package
```yaml
flutter pub get
```
- Run source code
```yaml
flutter run
```
